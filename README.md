# ZF-spider
正方教务爬虫
# 技术栈
 js+nodejs+Finder4
#爬坑

 -正方教务有一个开发人员登录页面（可以防止5秒刷屏），如果使用该入口，那么打开方式也会发生改变，原本的为window.open，而此入口是302重定向。
 -网上有很多教程，但是有一点没说清楚，你必须提交x-www-form-urlencoded数据。
 -正方教务系统使用字符串保存登录状态，经过测试大概15分钟，不同学校可能不同，登录后保存该字符串，无需再次登录减轻服务器压力
